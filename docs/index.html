Welcome to Conservator CLI’s documentation!¶

At its core, Conservator CLI is a Python library built to interface with Conservator. It also provides a nice CLI on top of the library.

Internally, Conservator CLI uses sgqlc to generate a Python API for Conservator. It then provides several layers of abstraction on top of this, to make operations as simple and easy as possible.

To get started, see the Installation guide. Then, get an overview using the CLI Quickstart or Python API Quickstart.
